# How-to-React-apps
1. download node.js

## - in terminal
2.**create react app**
```
 npx create-react-app app-name
```
if you are already in the folder type:
```
npx create-react-app ./
```
3. **start server**
 ```
npm start
```

## - in index.js file
1. to connect react app to DOM:
```
import React from 'react';
import ReactDOM from 'react-dom';

import App from './App';
```
